ผลงานที่ 1

การสร้างเว็ปเพจ


เว็ปเพจเป็นสื่อเทคโนโลยีที่เข้าถึงได้ง่าย ดังนั้นผมจึงได้เลือกการสร้างเว็ปเพจในการนำเสนอแฟ้มสะสมผลงานในอีกรูปแบบหนึ่ง โดยใช้แอปพลิเคชัน visual studio code ในการสร้าง และใช้ภาษาโปรแกรม 2 ภาษา ประกอบด้วย HTML ในการเขียนเว็บเพจหลัก และ CSS ในการตกแต่งหน้าเว็บเพจ ซึ่งเป็นการนำความรู้ที่ได้เรียนในมัธยมศึกษาตอนปลายมาสร้างเป็นผลงานชิ้นนี้

  

ผลงานที่ 2

การสร้างแอปพลิเคชัน


ปัจจุบันในการเรียนบางครั้งเรามีความจำเป็นที่จะต้องใช้เครื่องคิดเลข โปรแกรมแปลภาษา หรือ การจดบันทึกข้อมูลต่างๆ ซึ่งแต่ละโปรแกรมนั้นอยู่แยกกัน ด้วยเหตุนี้ผมจึงสร้างแอป SMART EDU ด้วย Thunkable ขึ้นเพื่อรวมแอปเหล่านี้เข้าด้วยกัน ให้เกิดความสะดวกในการใช้งานที่มากขึ้น

  

ผลงานที่ 3

โครงงานวิจัย


โครงงานวิจัย “IoT-Based Air Quality Monitoring System in Phuket, Thailand” เป็นการศึกษาเรื่องมลพิษทางอากาศในจังหวัดภูเก็ต โดยมีการใช้ IoT Sensor และติดตั้งที่ Supalai Resort, ท่าฉลอง และ เกาะเฮ โครงงานวิจัยนี้เป็นค่ายของโรงเรียน ทำให้ผมได้มีโอกาสรู้จัก IoT และเกิดความสนใจขึ้น

  

ผลงานที่ 4

การสร้างเกม


The Britishman in the Cave เป็นเกมที่ผมและเพื่อนๆ สร้างในการแข่งขัน HAMSTER HUB GAME JAME 4 โดยเป็นการสร้างเกมภายใน 5 วัน ภายใต้หัวข้อ “The old Game to the new World” ซึ่งผมได้ออกแบบเกมใน Level 2 และออกแบบส่วนหนึ่งขององค์ประกอบภายในเกม ในการแข่งขันนี้กลุ่มของผมอยู่ในลำดับที่ 6 จากทั้งหมด 11 กลุ่ม

  
  

ผลงานที่ 5

Pixel Arts


จากการทำเกม The Britishman in the Cave ทำให้ผมได้ลองฝึกการออกแบบ Pixel Arts เพื่อใช้เป็นองค์ประกอบภายในเกม ได้แก่ กล่องสมบัติ นาฬิกาทราย ประตู หน้าต่าง โคมไฟ ภายในเกม และนอกเหนือจากเกมนี้ผมยังได้ลองออกแบบ Animation ด้วยภาพ Pixel Arts ได้แก่ การหมุนเหรียญ และการเดินของนาฬิกา
